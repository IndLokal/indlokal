## Story Intent

Make the Section 17 operating model measurable with a narrow, decision-ready dashboard contract.

## Acceptance Criteria

1. Trusted Journey Resource Coverage is queryable.
2. Percent resources within TTL is queryable.
3. Stale exposure and outdated correction turnaround are queryable.
4. Anti-metrics are visible in release review.

## QA Gates

- Metric definitions are documented and sample-validated.
- Queries/readouts reflect actual product behavior.
- Dashboard contract is sufficient for go/no-go review.

## Engineering Path Focus

- Analytics event/readout layer
- Dashboard query contract

## Definition of Done

- Acceptance criteria met.
- Product/Ops can evaluate the track with live metrics.
