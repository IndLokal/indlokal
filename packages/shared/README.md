# @indlokal/shared

Shared API contracts and generated OpenAPI for the IndLokal monorepo.

- Zod schemas live in `src/contracts/<module>.ts`
- `openapi.yaml` is generated by `pnpm openapi:generate` (see [ADR-0002](../../docs/specs/ADR/0002-zod-as-contract.md))
- Both `apps/web` and `apps/mobile` depend on this package via `workspace:*`

Contracts are added incrementally as the corresponding TDD lands.
The first set comes with TDD-0001 (auth + /me).
